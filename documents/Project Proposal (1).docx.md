Project Title: Vision-Based Game Controller Using Hand Gesture Recognition

Short Description: 

This project aims to develop a real-time vision-based game controller that allows users to control a 2D computer game using hand gestures instead of a keyboard or mouse. A webcam will capture live video input, and a hand detection model (MediaPipe) will extract hand landmarks. These landmarks will be processed by a gesture classification module to recognize predefined static gestures such as fist, open palm, thumb up, and peace sign. Each recognized gesture will be mapped to a specific in-game action (e.g., move left, move right, jump, or stop). The system will focus on real-time performance, accuracy evaluation, and smooth interaction, demonstrating the integration of computer vision, machine learning, and interactive game development.
